# Attention-based AutoEncoder-like Orthogonal NMF for blind multispectral image decomposition

## Introduction
This code is a Tensorflow implementation of the Attention-based AE NMF model described in the paper here provided as a pdf, submitted for ICASSP 2025 :
**Attention-based AE Orthogonal NMF for blind image decomposition - Olive, Rahiche, Cheriet.pdf**

The used libraries are listed in `requirements.txt`.
